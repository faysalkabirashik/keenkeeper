# KeenKeeper

KeenKeeper is a relationship management web application where users can track friendships, manage interactions, and maintain meaningful personal connections.

## Live Site
(https://ph-keenkeeper.netlify.app/)

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- DaisyUI
- Recharts
- React Toastify
- LocalStorage

## Key Features

### 1. Smart Friendship Tracker
Track friends with overdue, almost due, and on-track statuses.

### 2. Quick Check-In System
Call, Text, or Video interactions instantly log into timeline with toast notifications.

### 3. Friendship Analytics
Visual pie chart dashboard showing interaction history.
